# Mew
Mew = Markdown Easy Wiki system

Markdownで記事を記述可能な完結なWikiシステムを目指して開発中です。

[PHP Markdown Extra](https://github.com/michelf/php-markdown)によってMarkdownをHTML化しています。
そのため、文法は基本的にPHP Markdown Extraに準じます。

## 機能
* ページの編集
* ページの追加
* ページの削除
* ファイルのアップロード
